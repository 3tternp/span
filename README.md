# Prem Basnet — Professional Portfolio

**Live site:** [basnetprem.com.np](https://basnetprem.com.np)

This portfolio presents Prem Basnet's 9+ years of cybersecurity expertise spanning:

- Vulnerability Assessment & Penetration Testing (VAPT)

## Getting Started

### Prerequisites

- Node.js 16+
- npm

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm start
```

Opens at [http://localhost:3000](http://localhost:3000). Hot-reloads on file changes.

### Build for production

```bash
npm run build
```

Outputs optimized static files to the `/build` directory.
